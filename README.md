# road-to-frontend
This repository documents my 6-day journey of practicing frontend development. Each day includes one mini project focusing on HTML, CSS, layout techniques, and responsive design.

## 6-Day Frontend Development Challenge

| Day   | Topic to Learn & Practice                     | What You Will Build                                         | Repo/Folder Name         | Commit Message Example                                |
|-------|-----------------------------------------------|-------------------------------------------------------------|--------------------------|--------------------------------------------------------|
| Day 1 | Basic HTML structure & tags                   | Personal Bio Page (name, photo, hobbies, links)             | `day1-personal-bio/`     | feat: added personal bio page structure                |
| Day 2 | CSS basics: fonts, colors, spacing, borders   | Styled version of your Bio Page                             | `day2-bio-styling/`      | style: improved UI, typography and spacing             |
| Day 3 | Flexbox layout arrangement                    | 3-card feature layout UI                                     | `day3-flexbox-cards/`    | feat: added responsive flexbox card layout             |
| Day 4 | Responsive design with media queries          | Mobile-responsive version of Day 3 layout                   | `day4-responsive-cards/` | feat: added mobile responsive breakpoints              |
| Day 5 | CSS Grid layout basics                        | Pricing Table (3 Plans) using Grid                          | `day5-pricing-grid/`     | feat: created pricing section using CSS grid           |
| Day 6 | Combine everything into one mini project      | One-page Landing Website (Hero, Cards, Pricing, Footer)     | `day6-landing-page/`     | feat: built complete responsive landing page           |
